# Drum Kit

Simple pure HTML, CSS and JavaScript drum kit that lets you play sounds with your keyboard.

## How to Run

1. Clone or download this folder.
2. Make sure all `.wav` files, `drum.png`, `index.html`, `css/style.css` and `js/app.js` are in the correct folders (see structure below).
3. Open `index.html` in your browser.

## Controls

You can play the drum kit with your keyboard:

- **A** → boom  
- **S** → clap  
- **D** → hihat  
- **F** → kick  
- **G** → openhat  
- **H** → ride  
- **J** → snare  
- **K** → tink  
- **L** → tom  

You can also click the buttons with your mouse.
